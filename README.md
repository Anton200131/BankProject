# Проект: Банковский сервис (Stage 1)

REST API приложение для управления балансом пользователей.

## Функциональные возможности
- Получение текущего баланса пользователя по его ID.
- Пополнение счета на указанную сумму.
- Снятие средств со счета (с проверкой на наличие остатка).

## Стек технологий
- Язык программирования: Java 17
- Фреймворк: Spring Boot 3.2.2
- База данных: PostgreSQL 17
- Сборка проекта: Maven

## База данных
В проекте используется таблица "users" со следующими полями:
- user_id (тип BIGINT, первичный ключ)
- balance (тип NUMERIC, хранение денежных средств)

Для воссоздания структуры базы данных используйте файл db_dump.sql, приложенный к проекту.

## Инструкция по запуску
1. Склонируйте репозиторий с кодом.
2. Создайте базу данных в PostgreSQL и импортируйте структуру из файла db_dump.sql.
3. В файле src/main/java/org/example/DatabaseManager.java укажите ваш пароль от локальной базы данных.
4. Запустите приложение через класс BankApplication.
5. Для проверки работы используйте адрес http://localhost:8080/api/balance/1 (в браузере или через curl).
